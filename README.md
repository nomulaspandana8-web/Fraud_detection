# 💳 Real-Time Fraud Detection System

## 📌 Overview
This project detects fraudulent online transactions using Machine Learning and Streamlit.

## 🧠 Technologies
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn

## 📂 Files
- app.py → Frontend
- model.py → ML model
- utils.py → Preprocessing
- creditcard.csv → Dataset

## ⚙️ How It Works
1. Dataset is loaded
2. Model is trained
3. User enters data
4. Prediction is shown

## ▶️ Run
pip install -r requirements.txt  
streamlit run app.py  

## 📊 Dataset
Credit Card Fraud Detection Dataset (Kaggle)

## 🎯 Output
- Fraud / Legitimate
- Probability score
